   | 功能                      | 快捷键                               |
   | 新建文件or打开文件        | c-x c-f                              |

   | 添加有名称的超链接        | [[url] [name]]                       |

   | 关闭当前分屏              | c-x 0                                |
   | 只保留当前分屏            | c-x 1                                |
   | 打开最近使用的文件buffer  | c-x c-r                              |
   | 删除buffer                | c-x k                                |
   | 切换到其他buffer          | c-x b                                |
   | 有道词典查单词            | 选中单词后，c-c o                    |

** 🐓日程提醒功能
   1. 删除 ~/.emacs.d/gtd.org
   2. c-c r打开添加事项buffer, 选择t(to do)，添加事项
   3. shift + 上下按键调节任务优先级
   4. c-c c-c保存，c-c c-k退出不保存
   5. c-c a进入agenda buffer,选择a查看当前用户日程
